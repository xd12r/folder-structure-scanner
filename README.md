# Folder Structure Scanner

A simple tool to scan a folder and display its structure in a tree format.

### Example Output
```
new-project/
├── Dockerfile
├── app/
│   ├── __pycache__/
│   │   ├── app.cpython-39.pyc
│   ├── app.py
│   ├── static/
│   │   ├── favicon.ico
│   ├── templates/
│   │   ├── create_note.html
│   │   ├── login.html
│   │   ├── note.html
├── docker-compose.yml
├── requirements.txt
```
